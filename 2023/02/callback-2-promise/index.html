Callback 转 Promise

发表于 2023-02-23 更新于 2023-06-07 分类于 技术积累 阅读次数：

情景

遇到有callback调用的函数，容易造成代码嵌套。

解决方法

node.js 8.0.0 版本中新增的一个工具 util.promisify，用于将有 callback 的函数转成返回 promise 的函数。

条件

  1. callback 是 error-first 模式、
